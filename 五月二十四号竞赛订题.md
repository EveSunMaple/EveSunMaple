
## Problem A. 硬币游戏 (coin.c/cpp)

### 题目描述

你的面前有 $n$ 堆硬币，第 $i$ 堆有 $A_{i}$ 个，你有一个宽度为 K 的勾爪，可以同时取走 $A_{i}$ 和 $A_{i}+K$，但是不能空钩，即 $A_{i}$ 和 $A_{i}+K$ 两堆都未被取走时才可以抓取。现在你可以进行任意多次抓取，请你求出你能获得的最大硬币数。

### 输入格式

第一行两个整数 $n$, $K$，表示硬币的堆数和勾爪的宽度。

第二行 $n$ 个整数 $A_{1}, A_{2}, . . . , A{n}$, 依次表示 $n$ 堆硬币的硬币数。

### 输出格式

输出一个整数，表示能获得的最大硬币数。

### 输入样例

|coin.in|coin.out|
|:---|:---|
|3 1 <br> 1 2 3|5|
|5 2 <br> 1 2 3 2 1|8|

### 解题思路

这道题目看起来像是一道动态规划题，但是仔细分析后我们会发现动态规划算法并不适用于这道题目。因为一个硬币只能抓一次，后面的转移可能会影响前面，所以我们需要使用贪心+分治算法来解决这道题目。

我们可以将硬币按照位置模 $K$ 的余数分成 $K$ 组。我们将硬币按照位置模 $K$ 的余数分成 $K$ 组后，每一组就相当于是 $K = 1$ 的情况。也就是说，我们可以将原问题转化为 $K$ 个 $K = 1$ 的子问题，然后分别求解每个子问题，最后将所有子问题的答案加起来就得到了原问题的答案。

对于每一组硬币，我们都可以选择抓取或不抓取。如果这一组硬币的数量是偶数，那么我们可以把它们全部抓取；如果这一组硬币的数量是奇数，那么我们可以把除了最小的那个硬币以外的所有硬币都抓取。

>为什么这样做？

	这样写的原因是我们希望最大化抓取到的硬币数量。对于每一组硬币，我们都可以选择抓取或不抓取。如果这一组硬币的数量是偶数，那么我们可以把它们全部抓取，这样就能获得最大的收益；如果这一组硬币的数量是奇数，那么我们不能把它们全部抓取，因为勾爪不能空钩。在这种情况下，我们可以选择不抓取最小的那个硬币，这样就能获得最大的收益。

	举个例子，假设 K = 2，硬币数量分别为 [1, 2, 3, 4, 5]。那么我们可以将硬币分成两组：[1, 3, 5] 和 [2, 4]。对于第一组硬币，数量是奇数，所以我们选择不抓取最小的那个硬币（即硬币 1），抓取剩下的两个硬币（即硬币 3 和硬币 5）。对于第二组硬币，数量是偶数，所以我们可以把它们全部抓取（即硬币 2 和硬币 4）。最终我们能够抓取到的最大硬币数量就是 3 + 5 + 2 + 4 = 14。

>但我们还需要判断特殊情况：比如假设为现在有一组数据[2,1,3,4,5]，按照上面的解释应该去掉1，但是如果去掉1，就不能取到所有的数了，这应该怎么解决？

在代码中，变量 `p` 表示当前组中最小的硬币，`parity ^= 1` 这一行代码用来判断当前组中硬币的数量是奇数还是偶数。`parity` 变量的初始值为 0，每次遍历到一个硬币时，都会执行 `parity ^= 1` 这一行代码，相当于将 `parity` 变量的值在 0 和 1 之间切换。

1. 如果 `parity` 变量的值为 1，那么说明放弃当前硬币可以保证剩下的硬币都能被取到，我们把它与 `p` 比较 。
2. 如果 `parity` 变量的值为 0，那么说明放弃当前硬币可以并不能保证剩下的硬币都能被取到，直接跳过。

最终，如果 `parity` 变量的值为 1，那么说明当前组中硬币的总数量是奇数，需要减去 `p`；如果 `parity` 变量的值为 0，那么说明当前组中硬币的总数量是偶数，无需特殊操作。

### 参考代码

```cpp
#include <bits/stdc++.h>

using namespace std;

int main() 
{
    int n, K;
    scanf("%d%d", &n, &K);
    vector<int> a(n);
    for (int i = 0; i < n; i++) scanf("%d", &a[i]);
    long long res = 0;
    for (int i = 0; i < K; i++) 
    {
        int p = 1e9, parity = 0;
        for (int j = i; j < n; j += K) 
        {
            parity ^= 1;
            if (parity) p = min(p, a[j]);
            res += a[j];
        }
        if (parity) res -= p;
    }
    printf("%lld\n", res);
}
```
